## 1. 交易系统总纲

### 少妇战法六步

完整交易闭环：①择时 → ②选股 → ③等B1 → ④设止损 → ⑤止盈放飞 → ⑥收队

### 执行口诀

B1买 → 设止损 → 破了平 → 涨起来BBI线上两根中阳线放飞一半 → 看BBI线 → 跌破两根全离场

### 正确交易 vs 错误交易

- 按照交易纪律做的单子，止损了也是正确单子
- 没按纪律做的单子，涨停了也是错误单子
- 交易不以盈亏论对错，以是否遵守纪律论对错

### 交易执行三段论

盘前选股（选方向）→ 盘中怎么买（敲单技巧）→ 盘后如何持有和止损（最重要）

### 复盘五步法

看新闻 → 看大盘 → 看热点 → 看持仓 → 选股+落实交易计划

### 模块化全景图

百课内容浓缩为一张三层模块图：**择时（红）→ 买入（蓝）→ 应对（黄）**，辅以分析依据（橙）。
- 择时是全部模块中最重要的
- 应对按优先级排列：没涨 → 盈转亏 → 止损 → 放飞 → S1 → 滴滴 → 破白线 → 破黄线 → 死叉
- 分析依据：关键K、暴力K、倍量柱、异动、ABC、N型、对称、三波、四块砖、填坑出坑、长音短柱
- 意义：新人快速定位学习路径，老手复盘交易在哪个环节出错

### 气弱留强

仓位管理原则：仓里只留强势图形的票，走坏形态的票拍掉。
- 走成弱势图形 → 拍掉
- 三天不涨 → 拍掉
- 强势图形 → 拿着
- **操作原则**：弱宝宝换强宝宝前，必须先找到强宝宝再切，不为了切换而切

### 三最原则（高阶交易框架）

完整交易链条的三环节：

**只选最美**（选股）→ **只买最强**（盘中）→ **只拿最硬**（持仓）

各环节权重：**交易纪律50% + 盘面30% + 选股20%**
- **只选最美**：选股只选完美图形，宁缺毋滥。"市场已经告诉你什么是顶美了"
- **只买最强**：盘中基于实时量比和量价配合，只买自选池里最强的。**只跟随不猜测**——这是灵魂
- **只拿最硬**：持仓里只留最强的，不行的拍掉换下一批。"好男儿志在四方，公主们全在包厢，不行就换下一批"
- **不给第二次机会**：能套你一次就能套你两次

### 双线战法（升级版交易体系）

从少妇战法（单线）升级为双线维度。

**白线（知行短期趋势线/牵牛绳）**：
- 公式：EMA(EMA(C,10),10)，双重指数移动平均
- 股价短期快速上涨期间一般不会跌破白线
- 破白线 = 短期趋势大概率走坏，全清
- **不是BBI**，但用法跟BBI类似

**黄线（知行多空线/大哥线）**：
- 公式：(MA(M1)+MA(M2)+MA(M3)+MA(M4))/4，四周期均线平均（默认M1=14/M2=28/M3=57/M4=114）
- 主力是否入场的标志性参考线
- 极限洗盘的极限位
- 大哥控盘的成本线
- 股价在黄线之下 + 白线之下 = 没有交易价值
- **不是MA60**，虽然接近但不等于MA60

**核心用法：**
1. 右侧只做白线在黄线上的票（白线金叉黄线 = 上涨趋势确立）
2. 短期放量站上白线才有上涨爆发力
3. 死叉未死叉 + N型结构不破 + 勾大幅值 = 极限买点
4. **N型上涨趋势中，白线与黄线之间的任何位置都是容错率较高的买入区** ← 最核心
- 白线死叉黄线 = 最后离场时机，走错也要走
- 横盘震荡期间（非N型）白线黄线没有概率优势，需切换成顶底支撑位

### 配置是最好的风控

"给一个股票十次机会，不如给十个股票一次机会"
- 大盘上涨时配置把仓位推上去，容错率高
- 大盘下跌时精选个股
- 养一池子鱼，哪条死了你都不知道

### 交易框架

每个人根据自身资金规模、心态、持股能力形成的个性化交易体系。不是复制别人的系统，而是找到适合自己的节奏。老师基于二十多年经验形成的框架，不适合直接移植给散户。
**原文金句：**"你有正反馈成功的那个感觉，就是你要记住那个感觉"

### ETF躺平策略

牛市强趋势阶段通过配置纯血ETF实现躺平盈利。
- **适用前提**：①牛市强趋势已开启（单边在BBI线上方）②增量资金不断 ③自己不会选股
- **熊市重个股、牛市重ETF**：熊市个股能跑超额，牛市ETF吃指数权重涨幅
- **纯血ETF**：前十大重仓占比高（如白酒ETF的67%仓位在白酒龙头）
- 做ETF要做波段，每个B1是一个波段

### 左侧尾盘再说 / 右侧开盘就干

- **右侧（B1/B2）**：开盘就干，基于盘前选股和量比
- **左侧（抄底）**：**尾盘再说**——当天不要开盘买，尾盘确认阳线 + 突破前高后再考虑

## 2. A股市场底层逻辑

### A股的定位：融资市场 → 投资市场

- A股成立之初的使命是为国企脱困服务（融资功能），而美股定位是全民共赢
- 过去20年指数3000点但活跃市值从8000涨到12万，钱流入了大股东和机构口袋
- 当前国家正推动从“融资市”向“投资市”的改革

### 国家意志决定论

- 大A核心驱动力是国家意志，不是基本面也不是情绪面
- “需要你的时候涨，不需要的时候别闹”
- 有形的手比无形的手牛逼100倍——中国制度优势
- A股底层认知六点：充分博弈、制度不对等、麒麟会、洋务派、政策驱动、波动盈利

### 政策周期

- 在中国最重要的择时要素，政策变动决定市场走势大方向
- 政策周期为经济体制改革服务
- 历史案例：05-07（国企改革）、09（救美国）、14-15（科教文卫）、当前（慢牛）

### 十年一梦周期论

- 每十年一次资产重新定价：第一年转势→一年横盘（消灭分歧）→主升浪
- 历史类比：2004→2007主升；2013→2015主升；2024（924）转势→当前横盘

### 168万亿存款理论

- 银行体系168万亿居民存款，来30万亿就能到6000点
- 场外资金极度充裕，缺的是信心和赚钱效应

### 反内卷逻辑

- A股新的宏观逻辑——国家政策推动反内卷，窗口指导级别
- 影响新能源车等竞争格局，利好龙头企业

### 防御板块底层逻辑：涨价

大盘调整时，化工、小金属、稀土等板块能作为防御品种的底层逻辑是**涨价确定性**。
- 涨价 = 业绩确定性强 → 市场调整时它能守住
- 反内卷是表象，本质是涨价（CPI转正需要良性通胀）
- 小金属很多是咱们国家控制的，涨价确定性高

### China New Story四大方向

科技→消费→医药→文化输出（下半年主线）。
主题共识形成路径：标签够多→标的够多→参与人够多→形成合力→容纳大资金→机构跟进→成为主线

### 顺周期启动顺序

银行 → 券商 → 科技 → 保险 → 消费

### “有人比你更着急”

- 国家层面比散户更希望市场好
- 八月重要会议前以稳为主

### 信息噪音鉴别

- 单条消息不改变整体策略，要看到真金白银的增量才能确认
- 新闻层面的利好不等于市场层面的机会

### 踏空鉴别框架

- **真踏空**：你懂的涨了你不在——心态问题
- **假踏空**：你不懂的涨了——正常

## 3. 择时体系

### 择时三要素

1. **基本面择时** → 判断大方向（宽松/紧缩，经济周期位置）
2. **资金面择时** → 看增量资金（活跃市值、Level2、国家队动向）
3. **技术面择时** → 找具体买点（K线形态、MACD、成交量）

展开为天时（宏观周期）、地利（政策周期）、人和（增量资金）框架。

### 活跃市值 / 0AMV

陈浩与指南针创始人共同发明，反映全A流通筹码总市值变化。阳线=增量资金入场，阴线=场内存量换手。来源：指南针免费版（guest登录）。

### 增量资金入场分级

| 级别 | 涨幅 | 性质 |
|:----|:-----|:-----|
| 小波段 | 4%+阳线 | 主题级别 |
| 大波段 | 5%-8%阳线 | 主线级别 |
| 十年一遇 | 7%-20%阳线 | 全市场普涨 |

### 2.3%离场法则

活跃市值单日跌幅≥2.3%→大概率区间见顶减仓。2.0%-2.2%艺术判断，1.7%以下正常回调。

### 增量资金 vs 存量资金

增量入场：活跃市值阳线，新钱买浮筹。存量切换：活跃市值阴线或小阳，场内资金主题轮动。

### 策略回测原则

策略必须分上涨/下跌/横盘三个区间分开测试，没有策略能在三种行情中都赚钱。

### 择时原则

- 宽松周期+经济拐点：不能空仓（至少半仓）
- 判断风险>收益：空仓也是策略
- 10年内不会再有924，不要抱不切实际幻想

### B1数量判断市场阶段

通过每日出现的B1数量变化判断市场所处的反弹阶段。

| B1数量 | 市场阶段 | 操作建议 |
|:-------|:---------|:---------|
| 900-700个 | 刚反弹 | 大量票可选，按板块筛 |
| 300-200个 | 反弹中段 | 精细筛选，找完美形态 |
| 200-120个 | 反弹尾部 | 还能玩3天左右 |
| 120个以下 | 反弹尾声 | 谨慎，不好做了 |

- **核心理解**：大盘每次调整都是市场资金切换的行为，主力在一致性认知时出货，去扒拉散户少的低位票
- **原文金句：**"当B1变少的时候，你手里如果拿的是三天前、四天前的B1，一定要研究你这是不是强壮的宝宝"

### 节奏仓位管理

根据每日钩数（B1数量/金叉数）动态调整仓位和选股标准。
- 钩多→多做，仓位重；钩少→少做，标准严
- 完美度标准随钩数递减而提高

| 每日钩数 | 操作 | 选股标准 |
|:---------|:-----|:---------|
| 2000+ | 买4个票，重仓 | 中等形态即可 |
| 1000+ | 买1-2个，中等 | 良好形态 |
| 700 | 买1个，轻仓 | 优秀形态 |
| 300 | 观察，极轻或空 | 只选完美图形（80分+） |

- **原文金句：**"节奏很重要"
- 仓位自然调整法：上面利润多的票放飞后，仓位自然缩到半仓，下一波再干

## 4. 选股与配置

### 选股逻辑链

基本面看政策 → 政策看行业 → 行业看龙头 → 盘面语言确认

### 核心原则

- **不离开主线**：每个阶段市场有明确主线
- **不碰下跌趋势板块**（大爷图/大野图）

### 主题投资 vs 主线投资

主题：时间短、资金容量小、波动大。主线：时间长、资金容量大、走势稳健。主线容错率>主题容错率。

### 池子（替代曼城阵容）

不再用曼城阵容固定比例配置模式，改为给出方向池子让学员自己选。
**结构：** 主线（如AIDC算力芯片存储）→ 支线（如新能源/化工新材料）→ 答应（创新药，主线支线熄火时补位）
**意义：** 更灵活简单，降低跟单难度

### 曼城阵容（大资金/中线配置）

4231阵型：40%低波红利防守 + 30%前锋进攻 + 20%中场过渡 + 10%机动备用。
每个票有明确角色（攻/守/对冲）。必须配置不单调。

### 机构资金定价权排序

国家队→社保→保险→公募→券商自营→私募→麒麟会→游资→散户→外资

### 配置平滑理论

曼城+主题的双层配置结构，长线+短线互补。

### 体量决定玩法

不同资金体量有不同策略。少妇战法支持到1000万，大资金做增发/并购。

### 填坑出坑为核心战法

波段操作=挖坑和填坑的循环。先填小坑再填大坑。不做C位选手，等主力填完一个小坑再入场。

### 坑不能做三条件

没填坑/不懂基本面/不在主线的坑不要碰。

### 出坑回踩确认

左右坑沿对齐，回踩不破50%即确认。大盘不突破时填大坑回踩不碰。

## 5. N型结构与节奏

### N型结构（核心）

价格以N型结构运行。低点抬高+高点抬高=上涨N型；高点降低+低点降低=下跌N型。
本质：不断抬高市场对手盘的跟风成本。
适用范围：所有走势都是N型结构（上涨/横盘/下跌）。

### 交易节奏三层体系

市场的节奏→个股的节奏→自己的交易节奏

### 吸呼节奏感知法

放量=吸（建仓），缩量=呼（洗盘）。
呼吸三依据：①向上N型结构 ②量能配合 ③KDJ确认。三个缺一不可。
呼吸紊乱：上涨缩量、下跌放量。

### 回撤 vs 反弹

上涨中的阴线叫回撤。下跌中的阳线叫反弹，到压力位必须出。

### 机构票 vs 麒麟会票 vs 游资票的呼吸区别

机构票呼吸顺畅规律（放量-缩量循环），麒麟会票大开大合+有顶有底震荡，老庄股缩量拉升，游资票无法预测。

### 上涨节奏终结判断

上涨趋势终结前必先走成震荡区间→破前低→确认下跌。

### 异动（移仓异动）

主力开始介入某只票的早期信号，通过量价配合识别。
- **初期的异动是最好的介入时机**——主力刚入场，还没拉升
- **堆量异动**：连续多根阳线带着温和放量，一坨一坨堆上去（建仓信号）
- **三连跳建仓**：三根连续的堆量阳线形成建仓信号
- **堆量后等B1**：堆量→缩量洗盘→B1是最佳买点

完整流程：异动（来人了）→ 堆量（建仓）→ 缩量（洗盘）→ B1（买入）→ 拉升（放量）→ 顶部放量（出货）

### 势能降低

股票上涨过程中，上涨力度在衰减的信号。
- **特征**：K线开始松散，底下的获利筹码开始出货
- 连续B1往下走很多，说明势能在降低
- 处理：注意防卖飞，该减仓减仓

### 完美图形评分体系

用NBA球员类比股票图形的"身体素质"，凭"身体素质"评分选股。

| 级别 | 特征 | 可做？ |
|:----|:-----|:------|
| 徐杰 | 单薄、量能弱 | 勉强，但有更好的就别做 |
| 胡明轩 | 合格，有一定厚度 | 可以 |
| 拉希德华莱士/本华莱士 | 还行到不错，有上限 | 可以做 |
| 杜兰特 | 优雅标准教科书 | 高分，大胆做 |
| 詹姆斯 | 顶级完美 | 满分，做错也认 |
| 蔡恩 | 不讲道理、暴力 | 不好说，看运气 |

- 完美图形的关键特征：呼吸同呼吸共命运、红肥绿瘦、缩量到极致、无大绿棒子
- **不完美**：价格涨量在跌、单薄、阴阳不调、层峦叠嶂
- **原文金句：**"就怕货比货"——见过更好的就知道差的不能干

### 展开

在一波多头区间内，各板块和个股有自己的节奏，不必然与大盘同步。只要大盘没完、增量资金还在进，几乎所有票都会走完自己要走的趋势。
**原文：**"行情会慢慢展开"

### 对称交易理论

走势天生具有对称性，是老师自创的核心交易理念。
- **走势B对称**：走势的左半边和右半边会形成对称关系
- **空间对称**：上涨对应下跌（直接对称/A字），或未完全对称（间接对称/回调一半）
- **量能判断**：对称走势中量守恒还是不守恒是判断关键
- 加速上涨时不能有大回撤，否则会走对称下跌
- **应用**：对称用于判断交易价值的延续性。SB1是对B1的对称延续

## 6. 量价与筹码

### 缩量真实·放量可作假

缩量做不了假：N型上涨找极致缩量点，N型下跌找放量点。放量可造假（左兜换右兜）。

### 3/4阴量线

（07-manual修正：原“4分之3音量线”）成交量萎缩到前一天的75%左右的阴线。不好信号。
对比：缩半量（50%）=好信号。

### 顶部无量

顶部无量=健康，主力没有在顶部出货。阳量不看量，阴量才看量。

### 顶部大风车

（07-manual修正）单日K线特征——放天量+长上影+长下影+螺旋桨。特指一根K线。见顶信号，走错也要走。

### S1/S2/S3 逃顶体系

加速行情中的三步逃顶信号系统：

| 卖点 | 信号 | 操作 |
|:----|:-----|:-----|
| **S1** | 加速后放巨量大阴线/长上影/顶部大风车 | 大票卖1/3-1/2，小票全卖 |
| **S2** | 双头创新高+MACD顶背离+量价背离 | 再卖部分 |
| **S3** | 反弹到放巨量K线下沿过不去 | 全卖 |

- **单顶不是顶**：大票必有双头或反抽，小票（300亿以下）可能直接A杀
- 不要在牛市追求精准逃顶——能精准逃顶说明交易系统不行，吃不满
- 卖点取决于买点：B1买的人以BBI持股，B2/B3买的人S1就要走

### 红肥绿瘦

阳量>阴量，健康持仓。

### 筹码纯度 / PURE筹码

麒麟会拿了70%流通筹码后，需让剩下30%成本控制在精确区间内。盘子越pure拉升越顺畅。

### 跟风盘成本交换理论

主力通过N型呼吸不断让低位跟风盘换到更高成本区间。跟风盘成本接近当前价格时洗不走。

### 底部放量破前高规律

月线底部区间成交量>前高底部成交量≈大概率破前高。

### 关键K线

放量K线、长下影K线、长上影K线。放量K线的高点=未来上涨的支撑位。

### 水母呼吸

K线成交量“一放一缩”的规律性节奏，健康形态。

### 五天换手率出货法

主力不用单日放天量，而是将一天要出的货分成5天匀速出完。
- 特征：连续5个交易日换手率累计100%+，每天看似没有放天量
- 对比：拉升时的缩量比例极小（只停1天），出货时平均成交量明显更高
- 核心：不需要放量，用时间换空间
- 案例：170亿盘子，5日成交156亿，换手率130%，已基本出完

### 麒麟会犬牙图形

标准庄股特征：K线图上密密麻麻的上影线和下影线交织，完全不讲技术逻辑。
- 处理：遇到直接略过，不看不碰
- 案例：鲁抗是标准庄股

### 长庄画图（兔耳朵）

有耐心的长庄主力会刻意画对称K线图形。
- 特征：顶部无量没出货，但K线形态刻意对称（如对称的兔耳朵）
- 含义：盘面语言显示主力有钱有耐心
- 处理：短期不要跟它墨迹

### 一波流票识别

一波流拉升不破BBI线、中间几乎不给上车机会的票。
- 特征：从底部一气呵成上去，不出现显著回调
- **不让你外盘散户进场**，买点没有研究意义
- 处理：不研究，换一批

### 走流畅的票一定不破BBI线

趋势流畅的票，主力绝不让自己图形难看。
- 特征：沿BBI线缓步上行，不破BBI线
- 含义：主力有节操，操盘规范
- 操作：BBI线上连续缩量上涨 → 防卖飞策略

### 箱体震荡老庄家

部分老庄股非常耐心，可以在一个箱体内震荡多年。
- 案例：迪迪从2013年箱体震荡到2020年，震了7年才拉起来
- 处理：只能跟它耗，没有耐心就别碰

### 阶梯量之后的SB1

大的阶梯量出现后不能马上做B1，需要足够长的时间消化。
- **标准阶梯量**：连续多根阴线逐级缩量，形状像楼梯
- 阶梯量后等SB1（超级B1）——时间要足够长
- 时间不够的阶梯量不做

### 一波流建仓拉合一

主力边买边拉，建仓和拉升放在一起。
- 特征：BBI线上一直买，一边买一边拉，不破BBI线
- 拉升前可能给一个B1但机会很少
- 第一波没进去就算了，不追
- 跟堆量建仓的区别：一波流不给上车机会，堆量建仓会缩量给B1

### 大队长思维

确定性强、多概念加持的龙头票是所有资金最终的归宿。
- 各类概念（游戏/AI/算力）的徒子徒孙闹到最后，统一到平台型龙头结账
- 大资金来了没得选，只能配这类确定性标的

### 量比开盘决策体系

基于通达信默认量比的完整开盘决策树，用于"只买最强"的盘中执行：

| 量比类型 | 量比范围 | 开盘表现 | 判断 |
|:---------|:---------|:---------|:-----|
| 低量比 | 10以下 | 震荡 | 等回调慢慢买入 |
| 中量比 | 7-16 | 可上可下 | 量价同步则涨，不同步则观察 |
| 高量比 | 20以上+股价上涨 | 大概率上攻 | 真金白银，干 |
| 高量比 | 20以上+股价下跌 | 砸盘/假启动 | 稳一手，除非带量反包 |
| 中高量比 | 10以上+低开2%+ | 飞刀 | 别接，观察

## 7. B1买点体系

### B1（第一买点）

条件：连续下跌→N型转折→KDJ大负值（J值<13可关注）→缩量确认。
B1不是抄底，是确认转折后才介入。
配合MACD底背离+BBI向上，确定性更强→做错也要做。
一般第二天开盘买入。

### B1精髓

买在短期获利盘/卖盘枯竭的地方，短期向下动能已释放。

### B1赚什么钱

赚N型结构填坑到第一波高点的钱。

### B1解决三大问题

①解决一买就被套 ②追求最大盈亏比（向上10%向下2%=5:1）③保持流畅节奏

### 买入三问

①买入预期是什么？（到前高？突破？填坑？）
②盈亏比划不划算？（短线至少3:1）
③哪里设止损？（前低/N型最低点/入场K线最低价下方）

### 短线三天不涨拍掉

B1买入后三天没脱离成本区，拍掉换下一批。

### B1三天原则（数据回测支撑）

B1买入后3天内起来就起来了，第4天开始概率衰减。
- 原文金句："三天之内起来就起来了，熄火就熄火了，三天之内的概率，庞大的概率是最高的"
- 与"短线三天不涨拍掉"一致，但补充了数据回测支撑

### B1数量与选股精细度

每日B1数量越少，选股越要精细。
- 700个B1：按板块筛选即可
- 193个B1：仔细扒拉每个，只能选出4-5个可干的
- 板块B1集中度法：当某板块集中出现大量B1（如71个医药），该板块大概率出牛股

### 不要把交易性机会当长线

破了止损位就走，不改成“不得不改的长线”。

### B2（第二买点）

确定性更强，但止损幅度更大（7%-8%）。未熟练掌握B1止损前不建议做B2。

### B3（第三买点）

翻倍票第三个B1假破位往往是深蹲起跳的真机会。

### SB1（超级B1）

定义：缩量横住→放量下跌（分歧）→又横住缩量→勾大幅值。
特征：图形最难看的时刻，大多数人在此被吓走。SB1不是主动等的，是“忍出来的”。
麒麟会洗盘博弈：主力借假破位把混进来的大资金筹码洗出来，快速拉回不给低位机会。

### B1/B2/B3三件套

同一波趋势中三个不同的确认买点。三件套凑齐 → 大概率沿白线流畅上涨。
- **B1**：KDJ勾大幅值后拐头阳线（底部N型转折）
- **B2**：B1之后确认趋势的中长阳。KDJ勾明显往上拐头
- **B3**：B2之后途中中继的小阳线。**99%的人不要做B3**

| 买点 | 盈亏比 | 上涨确定性 | 适合 |
|:----|:------|:-----------|:-----|
| B1 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | 所有人 |
| B2 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 有经验者 |
| B3 | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 高手（反应快） |

### 翻倍票B一操作规则

第一个B1不做，第二个B1稳一手，第三个B1破位是真机会。

### 回调一半法则

翻倍票回调到涨幅一半+缩量+钩大幅值=高质量买点。翻倍票第一个B1慎做。

### 接力图形

底部连续放量阳线（2-3根）+顶部缩量=机构建仓完毕。回调到B1位置介入。

### 合并同类项

多根K线合并。双B1/连续B1可合并为一个更大的B1区域。

### 洗盘识别

特征：股价没怎么跌但KDJ快速下到大负值→洗盘不是出货→B1买入良机。

### 假破位识别

破坑沿位不破低位，是主力洗盘手法。

### B3（系统版）

B1→B2之后完整波段可能走出的第三买点。特征：
- B3的砖形图稳定性比前两个低
- 能走出B3配合砖形图的就是大牛
- **B3只看白线（牵牛绳）**：走出B3后不看砖形图，钉白线跟着趋势走
- 等待老师后续补课系统讲解

### 砖形图嫁接

砖形图可嫁接在B1/B2/B3/单针下30等任一买入方式中使用，也可单独使用。作用是**再确认**当前买入是否有效：
- B1/B2买入后看砖形图是否红着→红着继续持有，绿了赶紧走
- B3的砖形图稳定性低，不做主要参考

## 8. 入场方式与战法

### 暴力特噗（突破入场）

条件：放量大阳线突破前高，解放所有套牢盘。特噗后不能出现3/4阴量线。

### 深水炸弹（超跌反弹）

DMI（ADX）指标拐头向上。需严格止损，3天不涨必走。仅日线级别，短线策略。

### 单针下20（知行洗盘短线）

通达信副图/选股公式（07-manual确认代码）。

### 补票战法

第一买点（B1）错过后的系统补票方案。突破后入场，适用于活跃市值红柱持续的市场环境。

### 建仓波B1买入

底部横盘后连续中长阳（25%-50%），不放涨停。缩量回调B1可靠性高。建仓波不能有涨停。

### 拉升波B1买入

快速脱离成本区，涨幅50%-100%。急跌可赌B1，缓跌等第2/3个B1。

### 冲刺波后处理

冲刺波顶部放量后B1一般不做。如果指数持续上涨三波会循环。赌第三波必须设近止损（2%-3%）。

### 三波理论循环图

建仓波→（缩量回调B1）→拉升波→（回调第2/3个B1）→冲刺波→（顶部放量离场）
↓如果指数持续上涨→回到建仓波→拉升波→冲刺波循环

### 长音短柱

K线实体较长但成交量很小。灾后重建最牛的票都是长音短柱下来后顺势起，是重要判断依据。

### 消息为盘面服务

新闻消息是为盘面走势服务的，不是盘面跟随消息。先关注盘面，再去看消息，千万别因为消息去买票。

### 开盘先买一半+贴着黄线加仓

集合竞价后先推一半仓位，贴日内均价线加另一半。

### 特噗后3/4阴量线判定

突破（特噗）后第二天缩量75%为3/4阴量线，不好信号。

### 涨停两阶段

涨停只在两种情况下出现：
1. **底部建仓阶段**：主力拿不到筹码，靠涨停吸引散户交筹码
2. **加速阶段**：趋势确认后涨停加速拉升

- 涨停后必须连续中长阳快速上拉才合理
- 涨停后出现三个上影线 → 等三天不涨拍掉

## 9. 止损体系

### 没涨（最重要的应对）

买入后没有快速脱离成本区，股价在成本线附近反复震荡、浮盈浮亏。**优先级高于止损。**
- 处理方式：拍掉换下一批
- **原文金句：**"市场没有证明你是对的时候，你就错了"

### 盈转亏

盘中浮盈转为亏损，必须立即处理。优先级仅次于没涨、高于止损。
- 处理方式：不等收盘，直接拍掉

### 只输一根K线（核心理念）

止损设在进场K线最低价下方几个价位（或1%），收盘跌破就走。永不套牢。

### 止损三级别

| 级别 | 设置方法 | 适用 |
|:----|:---------|:-----|
| 敏感型 | 进场K线最低价下方3-5个价位 | 新手/小资金 |
| 趋势型 | 本阶段最低点或上个阶段最低点 | 有经验者 |
| 盈亏比型 | 按盈亏比算（如向下10%向上50%=5:1） | 大资金 |

### 止损三原则（短线）

1. **赚钱的票不要做亏**：涨起来又跌回成本价，拍掉
2. **买入后该涨不涨拍掉**：成本区附近磨叽，拍掉（短线三天不涨拍掉）
3. **破掉止损价格拍掉**：收盘前确认，不盘中拍

### 中阳线标准（因票而异）

低波动票3%=中阳线，券商4%-6%，白酒3.5%-6%，创业板20cm 7%以上。

### 其他止损方式

N型止损、BBI拐头止损（BBI转下+2日收不回）、关键K止损。

### 每日压力测试

连续两个跌停不影响睡觉=最大仓位。

### 击穿对手盘

主力故意跌破黄线，将混入的大资金筹码洗出。处理方式：**忍一根**。
- 在跌破黄线第一根的最低价画横线
- 后续收盘跌破就走
- 没跌破、腻歪后收回来，就是击穿对手盘
- 忍一根是极限，不要忍两根

## 10. 止盈与收队

### 核心规则

BBI线上出现两根中阳线→减仓一半（卤煮/放飞）。剩余半仓以BBI线为持股依据。

### 第四块砖放飞铁律

砖形图出现第四块红砖时必考虑放飞，越往后越不稳定。第四块砖已有约10%浮盈，散户在此位置容易获利了结。
**放飞节奏：** 第一块砖少放→二三块起来再放→第四块放飞一半→出现S1再放一半

### 防卖飞策略

对于连续缩量上涨、趋势流畅沿BBI线运行的票，不轻易卖出。
- 适用条件：沿BBI线缓步上行，形态走流畅，主力不给难看图形的机会
- 执行：BBI线上横盘不破BBI线 → 拿住不动
- 切换时机：防卖飞策略只在脱离成本区后启用，未脱离时按"三天不涨拍掉"

### 模块化应对优先级

从最紧急到最终：**没涨 → 盈转亏 → 止损 → 放飞 → S1（五种出货方式） → 滴滴（阶梯量） → 破白线（牛股不破白线） → 破黄线（击穿对手盘，忍一根） → 死叉**

### 见顶信号

- **顶部大风车**：放天量+长上影+长下影+螺旋桨（一根K线）→走错也要走
- **MACD顶背离**：价格创新高但红柱缩短/黄线走低→必须减仓。**注意：背离只在放巨量之后才有判断价值**，没放量的顶背离在牛市里会背离再背离

### 收队规则

跌破BBI线两根K线→全仓离场。一切以收盘价为准。

### 出场原则

顶部形态宁可信其有。单顶不是顶（大盘股有二次冲高机会）。下跌中的阳线叫反弹。

### 上涨趋势横盘后的最后一笔

最后一笔B1操作一定会止损，接受这是交易的一部分。

## 11. 仓位管理

### 四不原则（行为风控）

不追 → 不动 → 不慌 → 不乱摸

### 动态仓（替代长线持有）

不做底仓、不做长线持有，全部做波段交易（动态仓）。大部分散户拿不住票，动态仓比长线更适合。**原文：**"别他妈弄什么拿住不动，拿不住，别吹牛逼，就按照波段做"

### 仓位纪律

- 100万以下分4-5个仓位
- 短线不加仓原则（没有脱离成本区前绝对不加仓）
- 同主题配3个以上标的组成组合
- 五六成固定仓位+浮动仓位
- 用输得起的钱交易
- 7%以上才叫崩，1.6%叫波动

### 两枪原则

一周开仓两次，降低交易频率。

### 三条内部纪律

不追高、不满仓单票、不扛单。

### N型结构连续B1加仓法

已有浮盈时二次B1可加仓。合并同类项双B1加仓。

### 短线不加仓原则

短线没有脱离成本区前绝对不加仓。

## 12. 市场参与者认知

### 机构 vs 游资 vs 量化

- **机构**：价值发现，不怕跟风盘，持续时间长。机构票底部连续中长阳不放涨停
- **游资**：击鼓传花，追涨停，靠隔夜溢价。游资言论的本质是培养韭菜接盘
- **量化**：趋势跟踪，专门收割游资。不要骂量化

### 跟庄三条件

让跟（主力全程不动）→蹭跟（箭在弦上时）→帮跟（B1帮锁仓）

### 建仓洗盘拉升出货四步模型

所有机构运作都遵循此四步周期。

### 公募行为框架

机构总是最后接盘（规则限制+业绩压力）。公募永远接盘。

### 游资言论认知框架

中高位涨停板都是出货。游资席位有时专门用来出货（折价5-10%大宗交易）。

### 麒麟会洗盘博弈

主力洗盘不是针对散户，是针对混进来的大资金。通过破位把多余筹码洗回来，然后快速拉回。

### 机构资金监测

- 国家队：通过央行每月非银行机构借贷数据追踪
- 券商自营：通过资管产品净值推断
- 麒麟会：完全不可见
- 游资：通过龙虎榜追踪

### 分歧与一致（量价关系）

缩量横住后突然放量下跌→短期分歧，分歧后缩量横住→重新一致，配合B1为右侧买点。

### 中线思维做少妇

不追求天天交易，做中线波段，耐心等待B1。

## 13. 交易心理与纪律

### 核心认知

- 机构思维：严格止损、配置、长线格局
- 盈亏同源：能单票赚大钱，就能单票亏大钱
- 积小胜为大胜：每次赚5%-10%，亏损控制在1%-5%
- 顺大势逆小势：做上涨趋势的回调，不做下跌趋势的反弹
- 踏空比亏钱更难受→永远要有一定底仓
- 永不套牢→不被套就有持续参与资格
- 永不炫技→大爷图不要碰
- “不要怕卖飞”：永远有下一批
- “永远有18岁的”：不恋战
- “你唯一的负责人是你自己”
- “选股不重要，执行最重要”
- “不要骂量化”：量化是游资的对手盘
- **“市场不是二元对立”**：交易不是对错黑白，是混沌的
- **“2万颗球”**：交易没有捷径，需要大量重复练习形成肌肉记忆
- **“你不能在市场好的时候才热爱”**：心态稳定的前提是接受市场的一切
- **“全职炒股不是长久之计”**：每年只有2个月多头区间，其他时间需要别的事做
- **不要补仓**：短线被套说明买错了，错了就不要补
- **每天翻1000个票，翻十年才能随便翻**：行情分析没有捷径，靠大量重复练习
- **785分之1的概率**：黑天鹅事件是交易的一部分，赶上了不怪你
- **降低预期，才不入急门**：一个月能做好一根就够了，不要对收益率有不切实际的幻想
- **牛市还没来**：现在是热身阶段，先把基础打牢
- **强者恒强**：市场永远是强者恒强，不敢买的会一直涨到所有人都相信。要做冠军粉，不要跟钱过意不去
- **不给第二次机会**：能套你一次就能套你两次，好股票不套人
- **物来则应，过去不留**：涨了按规则放飞，跌了按规则止损，不纠结
- **事儿反常必有妖**：主升浪中出现奇怪的K线（放量阴线/螺旋桨/接近跌停）→ 拍掉

### 交易六大人性问题

常见交易心态问题：五个是事实，一个是错觉。
1. **一追就跌，一卖就涨** → 事实。对N型结构和相对高低位置没概念
2. **死拿不涨，一卖就涨** → 事实。洗的就是你的对手盘
3. **多选一必选错** → 事实。你看懂的别人也看懂
4. **原本想买没买涨了，买了的跌了** → ❌ **错觉！** 认知不够，真看好早干了
5. **不敢买的涨，买的不涨** → 事实。强者恒强
6. **重仓不涨，轻仓涨** → 事实。好东西稀缺

### 成功才是成功的妈妈

看成功的案例建立信心，正向反馈。

### 市场跟随原则

跟随不是预判。市场没有证明你是对的=你是错的。

### 不要对股票有感情

所有票不过是一串代码。

### 常见错误

❌ 一跌就喊崩 ❌ 满仓单调 ❌ 追高 ❌ 短线思维做中线票 ❌ 对股票有感情 ❌ 把交易性机会当长线 ❌ 三天不涨不拍 ❌ 把盈亏当对错标准 ❌ 被套了还补仓 ❌ 没有强宝宝就乱切换 ❌ 研究一波流的票
