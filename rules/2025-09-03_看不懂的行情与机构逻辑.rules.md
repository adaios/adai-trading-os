## 规则 1：活跃市值连续大幅下跌时需警惕

- **适用场景**：大盘调整期的风险判断
- **规则内容**：
  - 当[[活跃市值 / 0AMV / 活筹指数]]连续两个交易日出现较大跌幅（单日跌幅超过2.3%），必须谨慎对待
  - 宁可信其有，不可信其无——视为[[增量资金离场信号]]
  - 观察次日能否收回，若不能收回则需要减仓防守
- **关联术语**：[[增量资金]]、[[增量资金离场信号]]

## 规则 2：牛市周线节奏规律

- **适用场景**：牛市中的周内节奏判断
- **规则内容**：
  - 牛市中以周线为准，周线保持向上
  - 牛市通常杀周二、周三，起周四、周五（确保周线向上）
  - 熊市则通常是周五砸盘（趋势向下，周末前出逃）
  - 投资者应适应这种节奏，不应在调整日中恐慌
- **关联术语**：[[A股时间分布规律]]

## 规则 3：机构资金分类与行为认知

- **适用场景**：理解市场博弈结构
- **规则内容**：
  - 机构资金分五类：某队 > 社保基金 > 险资 > 公募 > 各公司自营/委外
  - 险资做确定性波段，到目标即结账，不会因护盘需要而延迟
  - 公募只有在大量赎回时才会被迫卖出
  - 某队只托不举——大跌大托、小跌不托、关键点位托一下
  - 某队有成本控制，不会在高位接走所有抛盘
- **关联术语**：[[机构资金分类]]、[[麒麟会（机构主力）]]

## 规则 4：百岁山资金的识别与应对

- **适用场景**：区分市场资金类型以判断行情性质
- **规则内容**：
  - 百岁山资金 = 不相信中国经济能诞生全球性牛股的资金，习惯跟随美股（果链、达链等）
  - 百岁山踏空中国本土新消费/新经济后，会合力在海外映射方向抱团（如半导体）
  - 百岁山的进出可通过指数贡献排行榜的板块构成来识别
  - 散户策略：不做多头也不做空头，做滑头——跟着赚钱效应走
- **关联术语**：[[百岁山]]、[[指数贡献策略]]

## 规则 5：大盘回踩黄线是趋势延续的必要条件

- **适用场景**：大级别上涨趋势中的回调
- **规则内容**：
  - 任何一波大级别上涨趋势中，指数必须至少回踩一次[[黄线（大哥线）]]
  - 不回踩黄线的上涨无法走远——获利盘太重，不洗盘难以继续
  - 回踩黄线后的上涨才是真正可持续的上涨
  - 对未来可能打到黄线的回调做好心理准备和仓位管理
- **关联术语**：[[执行趋势线]]、[[战略回撤]]

## 规则 6：先到B1的票往往是弱势票

- **适用场景**：调整期选股
- **规则内容**：
  - 大盘调整时，最先回到B1位置的票往往是弱势票、没人护盘的票
  - 真正的强势票在大盘调整中仍然抗跌，不会先到B1
  - 选股应等待新的强势板块和标的从黄线附近启动
  - 第一波龙头的利润未了结前，下一波龙头不会出现
- **关联术语**：[[B1（买点1）]]、[[强者恒强]]、[[弃弱留强 / 去弱留强]]

## 规则 7：带量大阴线破位必须走

- **适用场景**：个股出现放量破位的大阴线
- **规则内容**：
  - 放大量的大阴线破位，说明大机构在真实卖出
  - 这种票不要抄底，不要幻想反转
  - 与盘中能收回的洗盘不同——带量破位的阴线是主力真实离场信号
  - 放量意味着筹码松动，后续需要很长时间修复
- **关联术语**：[[绿帽子（顶部放量阴线）]]、[[只输一根K线]]

## 规则 8：V型反转涨停 vs 破位不回——强弱识别

- **适用场景**：识别个股被错杀还是真实走弱
- **规则内容**：
  - 如果个股当天跌停但盘中快速收回并收盘翻红或接近翻红，属于错杀（假利空）
  - 如果个股大阴线破位后次日继续下跌且反弹无力，属于真实走弱
  - 错杀的票可以关注，真实走弱的票必须放弃
  - 判断标准：看反弹力度和成交量——缩量收回比放量收回更健康
- **关联术语**：[[主力洗盘（牛市多急跌阶段的洗盘）]]、[[假破位（洗盘破位）]]

## 规则 9：B1完美图形十张图方法——B1结业标准

- **适用场景**：对所有B1买点的标准统一
- **规则内容**：
  - B1选股标准化方法：记录10张标准完美图形作为"美女图"
  - 每天选股时，将候选票与标准图对比，符合标准才参与
  - 十张图背后的共同特征：[[建仓波]]放量启动 → 顶部无量 → 缩量回调 → 小阴小阳（振幅-2%到+1.8%）→ 勾到大幅值
  - 不符合标准图的B1不参与
- **关联术语**：[[完美图形]]、[[B1（买点1）]]、[[呼吸]]

## 规则 10：B1选股的振幅限制条件

- **适用场景**：B1选股的量价筛选
- **规则内容**：
  - B1候选票当日振幅需在7%以内（最佳4%以内）
  - 振幅过大说明多空分歧大、主力控盘弱
  - 勾大幅值标准：KDJ.J < 13（约等于）
  - 当日涨跌幅范围：-2% 到 +1.8%（小阴小阳区间的量化标准）
  - 必须是缩量+极致小K线（达成多空一致的标志）
- **关联术语**：[[B1（买点1）]]、[[分歧与一致（量价关系）]]

## 规则 11：分歧之后转一致才是买点

- **适用场景**：B1入场时机的精确判断
- **规则内容**：
  - 大阳线和大阴线代表多空分歧，不要在分歧时买入
  - 真正的买点是分歧之后转一致——多空达成平衡，成交量缩到极致的小阴小阳K线
  - 一致的标准：缩量 + 小振幅 + 小阴小阳
  - 从分歧到一致的过程是主力消化获利盘/清洗浮筹的过程
- **关联术语**：[[分歧与一致（量价关系）]]、[[关键K线]]

## 规则 12：牛市右侧ETF波段策略

- **适用场景**：牛市右侧趋势中的选股困难期
- **规则内容**：
  - 在牛市右侧上涨趋势中（白线与黄线金叉后），ETF也可作为N型结构交易的标的
  - ETF的N型结构B1同样有效，省去选股烦恼
  - 熊市和震荡市中不碰ETF（没有基本面信仰，被套后难解套）
  - 牛市结束后（S1/S2/S3完成后）也不再碰ETF
- **关联术语**：[[指数贡献策略]]
