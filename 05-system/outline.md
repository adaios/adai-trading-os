## 第一层：择时（什么时候做）

> ⭐ 全部模块中最重要的一层。择时不对，一切白费。

### 1.1 择时三要素
- 基本面择时 → 宏观周期
- 资金面择时 → 增量资金
- 技术面择时 → K线形态
- 展开为天时·地利·人和框架

### 1.2 活跃市值（0AMV）体系
- 0AMV 指标来源与用法
- 增量资金入场分级：
  - 小波段：4%+阳线 = 主题级别
  - 大波段：5%-8%阳线 = 主线级别
  - 十年一遇：7%-20%阳线 = 全市场普涨
- **2.3%离场法则**（-2.3 绿柱 = 减仓）
- 增量资金 vs 存量资金切换

### 1.3 择时原则
- 宽松周期+经济拐点：不能空仓
- 风险>收益：空仓也是策略
- 策略回测须分三区间（上涨/下跌/横盘）
- 10年内不会再有924

> **对应课程：** 0423、0514、0702

## 第三层：入场方式（怎么买）

### 3.1 核心买点体系

| 买点 | 条件 | 说明 |
|:----|:-----|:-----|
| **B1** | N型转折 + KDJ大幅值（J<13）+ 缩量确认 | 核心入场方式，不是抄底 |
| **B2** | B1之后的阳线确认 | 确定性更强，止损更大（7%-8%） |
| **B3** | B1→B2完整波段后 | 只看白线（牵牛绳），砖形图不稳定 |
| **SB1** | 缩量横住→放量下跌→再缩量横住→勾大幅值 | 主升前最后一洗，忍出来的 |

### 3.2 特殊入场方式
- 暴力特噗（突破入场）
- 深水炸弹（超跌反弹，ADX拐头）
- 单针下20（公式指标）
- 补票战法（B1错过后的补票方案）

### 3.3 买入三问（每次入场前问自己）
1. 买入预期是什么？（到前高？突破？填坑？）
2. 盈亏比划不划算？（短线至少 3:1）
3. 哪里设止损？（前低/N型最低点/入场K线最低价下方）

### 3.4 三波理论入场
- **建仓波B1**：底部横盘后中长阳（25%-50%），不放涨停
- **拉升波B1**：快速脱离成本区（50%-100%）
- **冲刺波B1**：谨慎处理，设近止损（2%-3%）
- 三波循环：指数持续上涨时循环接替

### 3.5 入场纪律
- 短线三天不涨拍掉
- 不要把交易性机会当长线
- 短线不加仓（脱离成本区前）

> **对应课程：** 0423、0430、0507、0514、0531、0601、0716、0415

## 第四层：分析依据（判断工具集）

> 支撑买入判断的工具，不直接产生交易信号，但决定买点质量。

### 4.1 N型结构与节奏
- N型结构（核心）：所有走势都是N型
- 交易节奏三层体系（市场→个股→自己）
- 吸呼节奏感知法（放量吸/缩量呼）
- 呼吸三依据（N型+量能+KDJ，缺一不可）
- 展开概念（行情慢慢展开，各走各的节奏）
- 上涨节奏终结判断

### 4.2 量价与筹码
- 缩量真实·放量可作假
- 3/4阴量线（不好信号，对比缩半量好信号）
- 顶部无量（健康），顶部大风车（见顶）
- 红肥绿瘦
- 筹码纯度/PURE筹码
- 跟风盘成本交换理论
- 底部放量破前高规律
- 关键K线（放量K/长下影/长上影）
- 水母呼吸（规律量价节奏）
- 长音短柱（K线长实体+小成交量）

### 4.3 砖形图（再确认工具）
- 砖形图六种定式
- 四块砖玩法
- 砖形图嫁接：可验证 B1/B2/B3 有效性
- 第四块砖放飞铁律

### 4.4 形态与结构
- ABC结构
- 对称结构
- 接力图形（底部放量+顶部缩量=建仓完毕）
- 回调一半法则
- 合并同类项
- 洗盘识别（股价不跌但 KDJ 大幅值）
- 假破位识别

### 4.5 指标系统
- MACD（顶底背离、线上金叉）
- KDJ（大幅值判定、勾）
- BBI（多空线、两根阳线放飞）
- ADX（深水炸弹拐头）

## 第五层：应对（买了以后怎么办）

> ⭐ 按优先级从最紧急到最终，逐级不跳。

```
没涨 → 盈转亏 → 止损 → 放飞 → S1 → 滴滴 → 破白线 → 破黄线 → 死叉
```

### 5.1 没涨（最重要）
- 买入后没脱离成本区，在成本线腻歪
- 处理：拍掉换下一批
- 金句："市场没有证明你是对的时候，你就错了"

### 5.2 盈转亏
- 盘中浮盈变浮亏
- 处理：不等收盘，直接拍掉

### 5.3 止损体系
- **只输一根K线**（核心理念）
- 止损三级别（敏感型/趋势型/盈亏比型）
- 止损三原则（赚钱回本拍→磨叽拍→破位拍）
- BBI拐头止损、N型止损、关键K止损
- 每日压力测试（两个跌停不影响睡觉）

### 5.4 止盈放飞
- BBI线上两根中阳线 → 减仓一半
- 第四块砖放飞铁律
- 放飞节奏：第一块少放→二三块再放→第四块放一半→S1再放一半

### 5.5 见顶出货信号
- S1（五种出货方式）
- 顶部大风车（放天量+长上影+长下影+螺旋桨）
- MACD顶背离
- 滴滴（阶梯量出货，高位连续两根阴线）

### 5.6 破线应对
- **破白线**：牛股不破白线，波段大概率结束
- **破黄线**：大哥不在，击穿对手盘·忍一根
- **死叉**：需要时间修复，一般人算了

### 5.7 收队规则
- 跌破BBI两根 → 全仓离场
- 顶部形态宁可信其有
- 单顶不是顶（大盘股有二次冲高）

### 5.8 应对核心认知
- **正确交易 vs 错误交易**：纪律论，无论盈亏
- **模块化应对优先级**：确保不跳级操作

> **对应课程：** 0507、0514、0709、0712、0716、0415

## 第六层：仓位与纪律

### 6.1 仓位管理
- 四不原则（不追→不动→不慌→不乱摸）
- 两枪原则（一周开仓两次）
- **动态仓**（替代长线持有，散户拿不住）
- 仓位纪律（分4-5仓、短线不加仓）
- 三条内部纪律（不追高/不满仓单票/不扛单）
- N型结构连续B1加仓法

### 6.2 交易心理
- 核心认知清单（积小胜为大胜、顺大势逆小势、永不套牢等）
- 正确交易 vs 错误交易（纪律论）
- 市场跟随原则（跟随不是预判）
- 不要对股票有感情
- 常见错误清单

### 6.3 交易框架（个人化）
- 每个人建立自己的框架
- "你不是我，你拿不住"
- "2万颗球"——肌肉记忆需要大量重复

### 6.4 复盘流程
- 复盘五步法（新闻→大盘→热点→持仓→选股+计划）
- 交易执行三段论（盘前选股→盘中买入→盘后持有和止损）
